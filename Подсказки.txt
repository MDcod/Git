git init - создание нового репозитория
git status - узнать статус репозитория
git log (--summary) - Просмотр сделанных коммитов (какие файлы добавлены/удалены/изменены)
git add fileName| *.txt | * - добавление новых файлов в Промежуточную Область
git reset - удаление файлов из Промежуточной области
git commit -m "Massage" - Добавление нового коммита с комментарием.
git remote add origin *url remote repo.git* - добавление удалённого репозитория
git push -u origin master - отправка коммитов на удаленный репозиторий (-u позволяет запомнить параметры и использовать дальше git push)
git pull origin master - взять коммиты с удаленного репозитория, просмотр изменений
git diff HEAD - сравнение полученного коммита от нашего последнего коммита
